# Notes CLI APP

## How to use

### Add a note

> node app.js add --title="\<your-title>" --body="\<your-body>"

### Remove a note

> node app.js remove --title="\<title-to-remove>"

### List all notes

> node app.js list

### Read a note

> node app.js read --title="\<title-to-read>"

### For more info

> node app.js --help
